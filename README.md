# SubSweep — دليل التشغيل والنشر

تطبيق كامل لتتبع الاشتراكات: تسجيل دخول حقيقي، قاعدة بيانات، واكتشاف تلقائي للاشتراكات المكررة وغير المستخدمة.

## 1. جهّز قاعدة البيانات (Supabase) — 5 دقائق

1. روح لـ https://supabase.com وسوّي حساب مجاني.
2. اضغط **New Project**، اختر اسم وباسورد لقاعدة البيانات (احفظه بمكان آمن).
3. بعد ما يخلص إنشاء المشروع (يأخذ دقيقة أو دقيقتين)، روح لتبويب **SQL Editor** من القائمة الجانبية.
4. افتح ملف `supabase/schema.sql` الموجود بهذا المشروع، انسخ محتواه كامل، الصقه بالـ SQL Editor، واضغط **Run**.
5. روح لـ **Settings > API** وانسخ قيمتين:
   - **Project URL**
   - **anon public key**

## 2. جهّز المشروع على جهازك

1. تأكد عندك Node.js مثبت (نسخة 18 أو أحدث).
2. افتح مجلد المشروع بالطرفية (Terminal) وشغّل:
   ```
   npm install
   ```
3. سوّي نسخة من `.env.example` باسم `.env.local`، وحط فيه القيمتين اللي نسختهم من Supabase:
   ```
   NEXT_PUBLIC_SUPABASE_URL=https://xxxxx.supabase.co
   NEXT_PUBLIC_SUPABASE_ANON_KEY=xxxxxxxxxxxxxxxx
   ```
4. جرب المشروع محليًا:
   ```
   npm run dev
   ```
   افتح المتصفح على `http://localhost:3000` — سجّل حساب جديد وجرب تضيف اشتراك.

## 3. انشره على الإنترنت (Vercel) — مجاني

1. ارفع المشروع على GitHub (سوّي repository جديد وارفع الملفات).
2. روح لـ https://vercel.com وسجّل دخول بحساب GitHub.
3. اضغط **Add New Project**، اختر الـ repository اللي رفعته.
4. قبل الضغط على Deploy، أضف نفس متغيرين البيئة (`NEXT_PUBLIC_SUPABASE_URL` و `NEXT_PUBLIC_SUPABASE_ANON_KEY`) من قسم **Environment Variables**.
5. اضغط **Deploy**. بعد دقيقة تقريبًا بيعطيك رابط جاهز مثل `subsweep.vercel.app`.

## 4. اربط دومين خاص بك (اختياري)

من إعدادات المشروع بـ Vercel > **Domains**، تقدر تضيف دومين اشتريته (من Namecheap أو GoDaddy مثلاً) وتربطه بخطوات بسيطة يوضحها Vercel نفسه.

## شنو ناقص عشان يصير منتج تجاري كامل؟

- **الدفع (Stripe)**: التطبيق الحالي مجاني الاستخدام بدون حد. لو تبي تخليه اشتراك مدفوع، أقدر أضيف صفحة تسعير + Stripe Checkout في خطوة لاحقة — تحتاج تفتح حساب Stripe فعلي أول.
- **ربط الحسابات البنكية تلقائيًا**: النسخة الحالية إدخال يدوي (وهذا مقصود ومتعمد للبداية). ربط بنكي فعلي (Plaid) يحتاج توثيق ومراجعة قانونية إضافية.
- **صفحة هبوط تسويقية (Landing Page)**: حاليًا التطبيق يفتح مباشرة على تسجيل الدخول. أقدر أبني لك صفحة تسويقية منفصلة لجذب المستخدمين.
